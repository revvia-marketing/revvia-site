# Card cover images

Drop a photo here to replace the **gradient box** on a card (the proof cards on
the homepage, the case cards on /work, and the production cards on /services).
Name the file after the card's slug — the part after the last `/` in its link:

| Card (where it appears) | Filename to drop here |
|---|---|
| EverSun/iTAN — Home proof + /work | `eversun.jpg` |
| Peter Grimm — Home proof + /work | `peter-grimm.jpg` |
| Voyage Air Guitar — Home proof + /work | `voyage-air.jpg` |
| Flagship Brand Film — /services | `flagship-brand-film.jpg` |
| Seasonal Content Capsule — /services | `seasonal-content-capsule.jpg` |
| Growth Foundation — /services | `growth-foundation.jpg` |
| Drop Launch — /services | `drop-launch.jpg` |

- Any of `.jpg .png .webp .avif` works (extension doesn't matter — match the name).
- Landscape shots ~1200px wide look best; they're shown behind the card's tag +
  name with a dark scrim for legibility, and run through the image pipeline.
- No file = the on-brand gradient stays. So add only the ones you have.

Upload via GitHub: this folder → **Add file → Upload files** → commit.
