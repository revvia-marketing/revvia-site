# Drop Launch — gallery photos

Drop photos here and they fill the gallery on **/services/drop-launch**,
automatically — optimized (responsive + modern formats), no code changes.

Filename controls order, size, and caption:

```
01-big-teaser.jpg      → first, large 2×2 tile, caption "Teaser"
hero-shot.jpg          → normal tile, caption "Hero Shot"
03-wide-drop-page.jpg  → wide tile, caption "Drop Page"
```

`NN-` sets order · `big` / `wide` / `tall` sets the tile size · the rest is the
caption. Types: `.jpg .png .webp .avif`. Until you add any, placeholder tiles
show. Upload via GitHub: this folder → **Add file → Upload files** → commit.
